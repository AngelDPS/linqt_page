Detailed Description

template<typename Tvalue>
class custom::vector< Tvalue, DeviceMemory >

Defines the vector class specialization for device memory type. This class is created using the CUDA C functions defined on cuda_runtime.h file. Therefore it is defined within the GPU_SUPPORT.
