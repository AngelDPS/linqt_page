Detailed Description

The tbmodel class This class is responsible for reading wannier files and converting it into a hopping_list structure It is also responsible for constructing other tight-binding operators

hopping_list tbmodel::createHoppingCurrents_list ( const int  dir)

Construct the current operator for the hopping list. In this context, a current operator is defined as {i,j} |i><j| I*H_{i,j}*(R_i-R_j)_dir with |i> the basis vector in which the hoppings <i|H|j> are defined. Here i represents both orbital and unit cell indexes. and d, representes the direction in which you want the current

Returns

Parameters
[in]Direction.The direction in which you want the current.
hopping_list tbmodel::createHoppingDensity_list ( oputil::op_matrix  OP)

Construct a density operator for the hopping list an a matrix defining an operator. In this context, the matrix is defined only in the unit cell, hereby the reason for calling it a density operator. The density operator is then defined as {i in cell} M_ij|i><j| with |i> the basis vector in which the hoppings <i,0,0,0|H|j,0,0,0> are defined. The (0,0,0) indexes is to note that this operator only consider hoppings within the unit cell

density_hopping_list Returns the density operator in the format of a hopping list.

hopping_list tbmodel::createLocalHopping_list ( )

Extract the local hoppings for the hopping list. In this context, the local hoppings are the one satisfying {i in cell} H_ij|i><j| with |i> the basis vector in which the hoppings <i,0,0,0|H|j,0,0,0> are defined. The (0,0,0) indexes is to note that this operator only consider hoppings within the unit cell
