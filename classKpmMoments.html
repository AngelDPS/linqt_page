Initialization using restart file.

It allocate the space for Mmax moments and look for the moments in a file called RestartInputName. If the number of moments in the file is higher than the requested it loads only the requested moments, if the number of the moments in the file is lower or equal to the requested moments, it will load all these moments. And if it doesn't find the file it will tag the simulation as and initial_simulation.
